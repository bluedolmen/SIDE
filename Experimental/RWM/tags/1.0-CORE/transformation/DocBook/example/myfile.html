Assistante

Documentation

Le groupe nommé Assistante regroupe l'ensemble des personnels attachés à un service administratif. Ce groupe a pour charge de stocker dans l'application l'ensemble des contacts obtenus pendant leurs différentes missions.

Est un agent humain : Oui

Responsabilité(s)

A la responsabilité de :

  • Saisir les informations ( voir la section intitulée « Saisir les informations » ) (Cet objectif se résume à la saisie d'informations.)

Télé conseiller

Documentation

Le groupe Télé conseiller est chargé de communiquer avec les différents contacts stockés dans les fiches mais son objectif est de maintenir une base d'informations à jour qui seront ensuite utilisée par les commerciaux.

Est un agent humain : Oui

Responsabilité(s)

A la responsabilité de :

  • Mettre à jour les informations ( voir la section intitulée « Mettre à jour les informations » ) (Cet objectif doit maintenir les données à jour afin qu'elles soient de meilleure qualité. Cet objectif facilite donc le travail du groupe Commercial.)

Analyste

Documentation

Le groupe Analyste de l'entreprise a pour tâche d'analyser les différentes communications avec l'extérieur et d'informer les dirigeants sur les différents profils existants dans l'ensemble des fiches. L'analyse se résume à une étude de la répartition géographique des contacts.

Est un agent humain : Oui

Responsabilité(s)

A la responsabilité de :

  • Analyser les informations ( voir la section intitulée « Analyser les informations » ) (Il est toujours nécessaire d'analyser les informations stockées par une application. Cette analyse permet d'avoir une meilleure vue sur les différents contacts et donc d'adapter les différentes prestations de l'entreprise afin d'améliorer ses performances.)

Administrateur

Documentation

Il est toujours nécessaire d'avoir un administrateur dans une application. C'est le seul profil qui a le droit de créer des carnets d'adresse et des répertoires.

Est un agent humain : Oui

Responsabilité(s)

A la responsabilité de :

  • Administrer les informations ( voir la section intitulée « Administrer les informations » ) (Cet objectif permet d'administrer les informations et donc de créer des carnets d'adresses et des répertoires.)

Commercial

Documentation

Le groupe Commercial a pour objectif d'utiliser l'ensemble des fiches saisies afin de vendre différents produits ou services.

Est un agent humain : Oui

Responsabilité(s)

A la responsabilité de :

  • Consulter les informations ( voir la section intitulée « Consulter les informations » ) (Cet objectif permet de consulter les informations de chaque fiche ajoutée dans l'application.)

Carnet d'adresse

Documentation

Le carnet d'adresse est le noyau de la structure d'information. Il est composé de différents répertoires.

Relations

Répertoire

Documentation

Un répertoire est un ensemble de fiches. L'ensemble des répertoires sont partagés dans l'entreprise. Un répertoire est caractérisé par son nom qui doit être unique.

Attributs

NomTypeDescription
nomTextualValue 

Fiche

Documentation

Une fiche est un ensemble d'informations concernant une personne. Elle regroupe son nom, son prénom, son adresse, son code postal et sa ville mais également son téléphone et son portable.

Attributs
